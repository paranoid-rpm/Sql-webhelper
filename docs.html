Справочник SQL-команд платформы

Раздел описывает команды, доступные в DBM Studio. Синтаксис ориентирован на практическую работу и покрывает DDL и DML-сценарии.

SELECT

Синтаксис Пример Комментарий
SELECT * FROM table SELECT * FROM users; Выбор всех колонок
WHERE + ORDER BY + LIMIT SELECT id,name FROM users WHERE id >= 2 ORDER BY id DESC LIMIT 2; Фильтрация, сортировка, ограничение
LIKE / IN SELECT * FROM users WHERE name LIKE 'А%'; Поиск по шаблону и спискам значений
JOIN SELECT users.name, orders.total FROM users JOIN orders ON users.id = orders.user_id; Соединение таблиц по условию ON
Агрегация + GROUP BY + HAVING SELECT user_id, SUM(total) AS revenue FROM orders GROUP BY user_id HAVING revenue > 500; Группировка и фильтрация агрегированных результатов

INSERT / UPDATE / DELETE

Операция Пример
INSERT (одна строка) INSERT INTO users (id,name,role) VALUES (4,'Олег','Viewer');
INSERT (несколько строк) INSERT INTO products (id,name,price) VALUES (504,'GPU',45000),(505,'NAS',87000);
UPDATE UPDATE users SET role='Senior DBA' WHERE id=1;
DELETE DELETE FROM orders WHERE total < 700;

Команда Пример Назначение
