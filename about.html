Архитектурная концепция

SQL WebHelper спроектирован как модульное веб-приложение, которое соединяет образовательный контент и рабочую панель администратора БД.

Слои приложения

  1. UI Layer: многостраничный интерфейс с единым дизайн-языком.
  2. Domain Layer: mini SQL-движок для CRUD-операций.
  3. Data Layer: in-memory модель таблиц как прототип реальной БД.

План расширения до production

  • Добавление backend API (Node.js/Python/Go) с настоящим MySQL.
  • JWT/OAuth2 авторизация и разграничение ролей DBA/Analyst/Viewer.
  • Миграции схемы через Liquibase/Flyway.
  • Набор шаблонов административных операций и отчётов.
