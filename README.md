# SQL WebHelper

Серьёзное, минималистичное и современное многостраничное веб-приложение для администрирования баз данных.

## Что внутри

- `index.html` — продуктовая главная с акцентом на DBA workflow, клиентскую ценность и сравнительную таблицу преимуществ.
- `theory.html` — расширенная теоретическая база по проектированию, оптимизации, эксплуатации и безопасности БД.
- `admin.html` — **DBM Studio**: in-browser симулятор администрирования таблиц с mini SQL и интерактивным гридом.
- `playbooks.html` — практические DBA playbooks для типовых инцидентов.
- `about.html` — архитектурная модель и roadmap.

## DBM Studio возможности

- Демонстрационная схема `demo_shop`.
- Просмотр таблиц `users`, `orders`, `products`.
- Интерактивная data-table:
  - сортировка по колонкам;
  - быстрый фильтр по всем полям;
  - KPI-виджеты (строки, колонки, история SQL).
- Поддерживаемые запросы:
  - `SELECT * FROM table;`
  - `INSERT INTO table (a,b) VALUES (1,'x');`
  - `UPDATE table SET a=2 WHERE id=1;`
  - `DELETE FROM table WHERE id=1;`
- История последних команд.
- Копирование системного лога.
- Сброс демо-БД.

## Дизайн и интерактив

- Минималистичный glassmorphism + серьёзная тёмная палитра.
- Canvas-анимация частиц на страницах.
- Плавные reveal-анимации блоков.
- Адаптивная структура для desktop и mobile.

## Запуск

Откройте `index.html` в браузере.
